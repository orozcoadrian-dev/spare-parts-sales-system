# Descripción

**JaoWorkshop** es un sistema de información desarrollado como proyecto académico de cuarto semestre, diseñado para optimizar los procesos administrativos y operativos del taller **Imperio Motos**, ubicado en la ciudad de Cartagena.

La plataforma busca digitalizar la gestión del taller, permitiendo un mejor control de clientes, motocicletas, órdenes de trabajo, servicios realizados y demás procesos internos, reduciendo el uso de registros manuales y mejorando la eficiencia del negocio.

---

# Objetivos

- Digitalizar la administración del taller.
- Optimizar el control de motocicletas y clientes.
- Facilitar el seguimiento de órdenes de trabajo.
- Centralizar la información del negocio.
- Mejorar la organización de los procesos internos.
- Proporcionar una solución escalable y fácil de mantener.

---
